# Gaoxuefeng's Blog

个人博客 - 分享技术、思考和生活体验

🔗 **博客地址**: https://gaoxuefeng.com

## 📚 内容

- 技术文章与学习笔记
- 代码片段与最佳实践
- 个人思考与经验分享

## 🛠 技术栈

- **静态生成**: [Hexo 8.1.1](https://hexo.io/) - 快速、简洁的静态博客框架
- **永链接**: [Abbrlink](https://github.com/rozbo/hexo-abbrlink) - CRC32 哈希生成的短链接 (避免中文 URL 编码问题)
- **图片加速**: [jsDelivr CDN](https://www.jsdelivr.com/) - 免费全球 CDN，支持国内访问
- **托管平台**: GitHub Pages
- **版本控制**: GitHub
